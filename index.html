zetas8.6 Shell 工具箱

功能强大的多功能命令行工具，集成了文件管理、系统监控、网络工具、Python环境等多项功能于一体，支持多命令执行和自定义扩展。

核心功能特性

zetas8.6 提供了丰富的功能，满足日常系统管理和开发需求

增强Python环境

内置增强的伪Python执行环境，支持变量持久化、模块导入和彩色输出，可直接在命令行中执行Python代码。

多命令执行

使用 + 符号分隔多个命令，支持一次性执行多个命令序列，提高工作效率。

颜色管理系统

完整的颜色控制功能，支持全局颜色设置、彩色文本输出，模仿PowerShell蓝色主题。

文件操作增强

完整的文件管理功能，包括复制、移动、删除、查找、查看等，支持带进度条的文件复制。

网络工具集成

内置ping、网络配置查看、网络连接状态等常用网络工具，无需额外安装。

系统管理功能

进程管理、系统监控、资源查看等功能，方便进行系统维护和故障排查。

使用示例

查看实际使用案例，快速上手 zetas8.6

多命令执行
# 使用 + 分隔多个命令 open 1.txt + open 2.txt + open 3.txt # 文件操作序列 cd C:\Users\Admin\Documents + find *.txt + cat report.txt # 系统信息检查 sysinfo + color_echo green:系统状态正常 + ps
Python环境
# 进入Python交互环境 /p # 在命令行中直接执行Python代码 /p print("$green:Hello, World!$") /p import math; print(f"圆周率: {math.pi}") # 计算阶乘 /p def factorial(n): return 1 if n <= 1 else n * factorial(n-1) /p print(f"10的阶乘是: {factorial(10)}")
文件操作
# 创建文件和目录 mkdir project cd project touch main.py utils.py README.md # 复制文件并显示进度 cp -p largefile.iso D:\backup\ # 查找特定文件 find *.py find *report*.txt # 查看文件内容 cat config.json type settings.ini
